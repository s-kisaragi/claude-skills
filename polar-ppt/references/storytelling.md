## Pyramid Principle

Barbara Minto's framework: start with the answer, then support it.

```
         [Main Recommendation / Answer]
        /            |                \
 [Argument 1]   [Argument 2]    [Argument 3]
  /     \         /     \         /     \
[Data] [Data]  [Data] [Data]  [Data] [Data]
```

**Key rules:**
- The governing thought (answer) comes first — as the action title
- Supporting arguments are **MECE** (see below)
- Each argument is supported by evidence
- If you read ONLY the action titles of all slides, they tell the complete story (the "elevator test")

**SCQA introduction pattern:**
1. **Situation**: Context the audience agrees with
2. **Complication**: The problem or change
3. **Question**: The implicit question this raises
4. **Answer**: Your recommendation (the governing thought)

## Action Titles

The **most critical element** of consulting presentations. Every content slide title must be a **conclusion sentence**, not a topic label.

### Rules

1. Must be a complete sentence with subject + verb + object
2. States the "so what" — the conclusion, not the topic
3. Maximum 2 lines (~120 characters)
4. Reading all titles sequentially must tell the story
5. Must be supported by the slide's content

### Examples

| Bad (Topic Title) | Good (Action Title) |
|---|---|
| Revenue Overview | Revenue grew 15% YoY, driven by Asia expansion |
| Market Analysis | The addressable market will double by 2028, creating a $4B opportunity |
| Cost Structure | Three cost categories account for 80% of the increase |
| Digital Transformation | Digital channels now drive 40% of new customer acquisition |
| Competitive Landscape | Company X is uniquely positioned to capture share from weakening incumbents |
| Project Status | Phase 2 is 3 weeks behind schedule due to vendor delays |
| Customer Satisfaction | NPS improved from 32 to 51 after redesigning the onboarding flow |

### Japanese Action Titles

Japanese action titles follow the same principle but adapt to Japanese grammar:

| Bad | Good |
|---|---|
| 市場概況 | 国内市場は2028年までに倍増し、4兆円の事業機会が生まれる |
| コスト分析 | 3つのコストカテゴリが増加額の80%を占めている |
| DX推進状況 | デジタルチャネル経由の新規顧客獲得が全体の40%に到達 |

Japanese decks may use 体言止め (noun-ending) for shorter titles:
- 「デジタルチャネル経由の新規顧客獲得率40%到達」

## MECE Principle

**M**utually **E**xclusive, **C**ollectively **E**xhaustive — every breakdown must:
- Have no overlaps between categories
- Cover the entire scope with no gaps

**Apply to:**
- Slide deck sections (each section covers a distinct topic)
- Bullet points (each bullet covers a distinct aspect)
- Chart categories (no overlap, no missing segments)
- Framework dimensions (2×2 axes must partition the space)

**Common MECE frameworks:** 3C (Customer/Company/Competitor), 4P (Product/Price/Place/Promotion), Porter's 5 Forces, Value Chain, PEST/PESTEL, Profit Tree

## Deck Structures

### Strategy Proposal (戦略提案書)

```
1. Title Slide
2. Executive Summary (1-2 slides: key recommendation + rationale)
3. Agenda
4. Section 1: Current Situation (2-3 slides)
   - Market context / industry trends
   - Client's current position
5. Section 2: Challenges & Opportunities (3-4 slides)
   - Key challenges identified
   - Market opportunity sizing
   - Competitive dynamics
6. Section 3: Recommended Strategy (4-6 slides)
   - Strategic options evaluated
   - Recommended approach
   - Implementation roadmap
   - Expected impact / business case
7. Section 4: Implementation Plan (2-3 slides)
   - Workstreams and timeline
   - Resource requirements
   - Quick wins vs long-term initiatives
8. Key Takeaways & Next Steps
9. Appendix
```

### Market Analysis (市場分析レポート)

```
1. Title Slide
2. Executive Summary
3. Agenda
4. Market Overview (2-3 slides)
   - Market size and growth
   - Key segments
5. Competitive Landscape (2-3 slides)
   - Player mapping
   - Market share analysis
6. Trend Analysis (2-3 slides)
   - Macro trends
   - Technology disruption
   - Consumer behavior shifts
7. Opportunity Assessment (2-3 slides)
   - Whitespace analysis
   - Attractiveness evaluation
8. Implications & Recommendations
9. Appendix
```

### Management Report / Quarterly Review (経営報告)

```
1. Title Slide
2. Executive Summary (1 slide: overall performance)
3. KPI Dashboard (1-2 slides)
4. Financial Performance (2-3 slides)
   - Revenue, profit, margins
   - Variance analysis (waterfall)
5. Operational Highlights (2-3 slides)
   - Key achievements
   - Challenges and mitigation
6. Strategic Initiatives Update (2-3 slides)
   - Project status (RAG)
   - Milestones achieved
7. Outlook & Priorities (1-2 slides)
8. Appendix
```

### Narrative Techniques

**SCR Arc** (Situation-Complication-Resolution): Main narrative flow for the entire deck.

**Breadcrumb technique**: Recurring agenda slide with current section highlighted, helping audience track position.

**Bookend technique**: Opening with the key question, closing with the answer — creating a satisfying narrative loop.

## Japanese Consulting Context

### Structural Differences

| Aspect | Western Style | Japanese Style |
|--------|--------------|---------------|
| Detail level | Less is more | Thorough detail expected (根回し) |
| Evidence | Key data points | Comprehensive evidence trail |
| Format | Action titles | Action titles + topic context |
| Appendix | Minimal | Extensive (shows due diligence) |
| Q&A prep | Verbal | Backup slides in appendix |

### Japanese-Specific Slide Patterns

**Issue-to-Initiative Mapping (課題→施策):**
Left column: issues (課題), Right column: corresponding initiatives (施策) with arrows connecting them.

**Evaluation Matrix with Japanese Symbols:**
Instead of Harvey balls, use ◎ (excellent), ○ (good), △ (fair), × (poor).

**Phased Approach (フェーズ別アプローチ):**
Timeline with Phase 1 / Phase 2 / Phase 3, each with deliverables and milestones. Common header: 短期 / 中期 / 長期.

### Bilingual Deck Considerations

When generating bilingual (EN/JP) decks:
- Primary language in main text, secondary in parentheses for key terms
- Or: parallel slide pairs (EN page followed by JP page)
- Japanese slides typically need ~20% more space due to character density
- Use Meiryo or Yu Gothic for Japanese body text
